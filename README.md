# tegaki_jan_rails

## Installation

Add this to your Gemfile:

```ruby
gem "tagaki_jan_rails"
```

and run `bundle install`.

## Usage

In your `application.css`, include the css file:

```css
/*
 *= require tagaki-jan
 */
```

### Helpers

```ruby
tj_icon 'east'
#=> <i class="tj tj-east"></i>
```
